# AI Short Drama Greenlight Committee Kit

English | [简体中文](README.zh-CN.md)

Greenlight committee templates for AI short drama projects: concept fit, audience promise, budget risk, production readiness, and release plan.

This repository is a neutral, practical resource for AI short drama greenlight review, concept approval, budget risk, production readiness, and release planning. It is designed to help teams make clearer decisions, write better review notes, and keep workflows reusable.

## Why This Repository Exists

AI short drama ideas need a clear greenlight process before production. A committee kit helps teams decide whether a concept is ready, what risk remains, and which evidence should be gathered before generating assets.

## Who This Is For

- AI short drama teams approving new projects
- producers reviewing concept readiness
- creators deciding which ideas deserve production resources

## Core Workflow

1. Define the review or production goal
2. Collect the required inputs and constraints
3. Use one template to make decisions comparable
4. Document fit, risks, and next actions
5. Refresh the workflow after real usage data
